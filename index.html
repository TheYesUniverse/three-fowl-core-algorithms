Welcome


Learning about algorithms while getting a handle on unit testing can be fun! This application was built for developers as a way to get familiar with how mocha and chai unit testing works in the browser.


  • Choose from 15 algorithms for a chance to learn a little about each and see how an implementation of said algorithm looks like in javaScript code.

  • On each algorithm you'll have the choice to 'add a bug' - a feature that simulates what a poorly written version of our code might look like.

  • Run a Mocha and Chai unit test right in the browser with a click of a button!

  • Enjoy.

Make Change Algorithm

Description

Given a price and an amount paid, the makeChange algorithm should return the number of coins in each denomination that should be given as change.


Code Implementation

                      
                       const makeChange = (price, amountGiven) => {
                         let amount = amountGiven - price;

                         let normalized = Math.floor( ( amount || 0 ) * 100 )

                       let quarters = Math.floor( normalized / 25 )
                         normalized -= quarters * 25

                         let dimes = Math.floor( normalized / 10 )
                        normalized -= dimes * 10

                         let nickels = Math.floor( normalized / 5 )
                        normalized -= nickels * 5

                         let pennies = normalized
                      }

FizzBuzz Algorithm

Description

Given any two integers fizzbuzz returns an array of the numbers between the given two numbers but returns 'Fizz' for multiples of three, 'Buzz' for multiples of five and'FizzBuzz' for multiples of fifteen.


Code Implementation

                  
                    var fizzbuzz = function(a, b){

                      for(var i = a; i <= b; i++){

                        if( i % 15 == 0 ){
                          console.log("Fizzbuzz");
                        } else if ( i % 5 == 0 ){
                          console.log("Buzz");
                        } else if ( i % 3 == 0 ){
                          console.log("Fizz");
                        } else {
                          console.log(i);
                        }
                      }

                    }

Palindrome Algorithm

Description

Given any word the palindrome algorithm returns a boolean value indicating whether the word passed is a palindrome or not.


Code Implementation

                    
                      const palindrome = (arg) => {

                        let submission = arg.replace(/[^\w,]/g, '');
                        let submissionArray = submission.split('');
                        let loopLength;
                        let middleNumIndex
                        let w;
                        let e;

                        if(submissionArray.length % 2 == 0){
                           loopLength = submissionArray.length / 2
                            w = (submissionArray.length / 2) - 1
                            e = w + 1

                        } else {

                            middleNumIndex = (submissionArray.length / 2) - 0.5
                            loopLength = (submissionArray.length / 2) - 0.5
                             w = middleNumIndex - 1
                             e = middleNumIndex + 1
                        }

                        for(let i = 0; i <= loopLength; i++){
                          if(submissionArray[w] == submissionArray[e]){
                              console.log('your entry: '+arg)
                              console.log(submissionArray[w]+' matches '+submissionArray[e])
                          } else {
                            console.log(submissionArray[w]+' does not match '+submissionArray[e])
                            return false
                          }
                            w -= 1
                            e += 1
                          console.log('true')
                        }

                        return true

                      }

Factorial Algorithm

Description

The factorial algorithm takes a non-negative number and returns the result of a factorial operation on that number.


Code Implementation

                  
                    const factorial = (arg) => {

                           if(arg == 0) {
                               return 1;
                           }
                           if(arg < 0) {
                               return undefined;
                           }
                           for(var i = arg; --i; ) {
                               arg *= i;
                           }
                           return arg;
                    }

Fibonacci Algorithm

Description

Given a positive number x the fibonacci algorithm returns special fibonacci numbers, ordered sequentially, in the amount of x.


Code Implementation

                    
                      const fibonacci = ( number ) => {
                        const fibArray = [0, 1]
                        if ( number <= 0 ) return undefined;
                        if ( number === 1 ) return fibArray = [0]
                        if ( number === 2 ) return fibArray = [0,1]
                        if ( number >= 3 ) {
                          for ( var i = 2; i < number; i++ ) {
                              fibArray[ i ] = fibArray[ i - 1 ] + fibArray[ i - 2 ]
                          }
                        }
                          return fibArray
                      }

Collatz Conjecture Algorithm

Description

This algorithm proves mathematician Luther Collatz conjecture that starting at any positive number n and taking the very specific trivial seeming steps (if n is even divide it by 2 to get a new value of n, if n is odd multiply it by 3 and add 1 to get a new value or n, and repeat ) it will always reach one.


Code Implementation

                      
                        const collatzConjecture = (start) => {

                          let array = [];

                          const doCollatz = (n) => {
                            if( typeof( n ) !== 'number' ) {
                              throw "InvalidInput"
                            }

                            array.push(n);

                            if (n === 1){
                              return 1
                            }
                            if(n % 2 != 0) {
                              n = (n * 3) + 1;
                              array.push(n);
                            }
                            if (n % 2 === 0) {
                              n = n/2;
                            }
                            doCollatz(n)
                          }

                          doCollatz(start)

                          return array
                        }

Run It


collatzConjecture();
