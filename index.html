Welcome


Learning about algorithms while getting a handle on unit testing can be fun! This application was built for developers as a way to get familiar with how mocha and chai unit testing works in the browser.


  • Choose from 15 algorithms for a chance to learn a little about each and see how an implementation of said algorithm looks like in javaScript code.

  • On each algorithm you'll have the choice to 'add a bug'. - a feature that simulates what a poorly written version of our code might look like.

  • Run a Mocha and Chai unit test right in the browser with a click of a button!

  • Enjoy.
